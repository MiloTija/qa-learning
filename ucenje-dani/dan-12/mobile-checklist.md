Zadatak 1 — Mobile Testing Checklist (45 min)

Napiši kompletnu Mobile Testing Checklist za imaginarnu e-commerce mobilnu aplikaciju.
Minimum 25 stavki u kategorijama:

□ Instalacija i update (5 stavki)
-> app se instalira bez greske
-> app se otvara nakon instalacije
-> install na punu memoriju
-> dozvole se traze na pravi nacin
-> update sa prethodne verzije funkcionise dobro 
-> update sa prethodnih par verzija funkcionise dobro

□ UI i rezolucije (5 stavki)
-> elementi su vidljivi na svim rezolucijama
-> text nije isecen
-> dugmici su dovoljno veliki za touch
-> sve je na adekvatnom jeziku (na jeziku sistema, ako je podrzano)
-> sadrzaj nije prekriven sistemskim dugmicima

□ Core funkcionalnost (5 stavki)
-> sve core funkcije rade
-> back btn/swipe funkcionise
-> deep links rade
-> login funkcionise
-> dodavanje u korpu funkcionise

□ Interrupcije (5 stavki)
-> dolazni pozivi- app se pauzira/nastavlja
-> push notifikacija - app ne crashuje
-> background/foreground tranzicija
-> app kill- sacuvane sve stavke
-> low battery upozorenje
-> battery saving mode on
-> headphones plug/unplug

□ Connectivity (5 stavki)
-> WiFi/ofline/WiFi
-> Slow conection handling
-> time out poruke
-> sinhronizacija nakon gubitka i ponovnog uspostavljanje mreze
-> 4G/3G/2G degradacija
